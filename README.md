# Gas Control

Sistema de gestão de fluidos refrigerantes focado na transição de gases poluentes (R-22) para opções de menor impacto ambiental (R-32).

## Sobre o Projeto
Este projeto gerencia um inventário de 50 equipamentos de ar-condicionado. O objetivo é fornecer visibilidade em tempo real sobre a utilização do gás R-22, auxiliando a equipe de manutenção no plano estratégico de substituição pelo gás R-32, que é menos nocivo ao meio ambiente.

## Tecnologias Utilizadas
* Frontend: HTML5, CSS3, JavaScript
* DevOps: Docker, Docker Compose
* Servidor Web: Nginx (Alpine Linux)
* Controle de Versão: Git e GitHub
